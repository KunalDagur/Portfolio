UI/UX Design

UI refers to the point of interaction between a user and a product. It includes the screens, buttons, toggles, icons, and other visual elements that you interact with when using a website, app, or electronic device. UI design focuses on the visual, interactive elements of a product to create aesthetically-pleasing interfaces.
